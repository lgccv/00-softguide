- ipdb (**python -m pdb main.py**)
  pip install ipdb -i https://mirror.baidu.com/pypi/simple
  1、**n(next)下一步**
  2、**b 14(14行下断点)**
  3、**s(step)进入函数体**
  4、**q(退出调试环境)**
  5、**p data(打印某个数据)**
  6、**run (运行到程序的下一行)**
  7、**where 查看调用堆栈**
